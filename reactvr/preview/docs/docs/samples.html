Samples #

The sample applications demonstrate various features of React VR. For setup instructions, see the Running the Samples.

  • Layout Sample: This simple scene displays a list of buttons on top of a panorama. It uses the flexbox layout algorithm that is built into React <View> to arrange the buttons vertically. Layout automatically applies to all view children, automatically computing their placement.
  • Mesh Sample: A basic scene that loads and displays a spinning model. To do this, an object model is loaded and placed in the scene with the <Mesh> tag; it is also illuminated with a <PointLight>. Rotation is achieved based on a React state variable that is applied as a rotation transform.
  • Cube Sample: Interactive scene that displays a button and geometric cube that changes color when the button is selected. Demonstrates use of the <VrButton> tag and of a Native Module. This sample uses the built-in cursor, enabled in vr/client.js.
  • Tour Sample: A complete app that implements a tour using linked 360 photos. The <NavButton> is activated by gazing or by direct selection, and moves between tour locations. <InfoButton> displays a <Tooltip> with text and/or images. Tooltip data and photo URLs are read from a JSON file. This app uses React Animations to provide fade-in and gaze-and-fill effects.
