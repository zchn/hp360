VrSoundEffects #

VrSoundEffects is a utility for playing audio clips, such as menu sounds when interacting with buttons or other UI elements. Note these are 2D sounds; for 3D/positional audio use the <Sound> component.

Audio clips must loaded before playing them. The argument to load is an audio resourse in the format {uri: 'http'} and the returned handle is a unique identifier that can be passed to play, stop, or unload. Use setOnReady to specify a callback to envoke when an audio clip is loaded and ready to play.

Example usage:

const sound = {uri: 'assets/sounds/my-sound.wav'}; const handle = VrSoundEffects.load(sound); VrSoundEffects.setOnReady(handle, function() { VrSoundEffects.play(handle); });

For another example of using this utility, see <VrButton> which provides sound effects triggered by interaction (click, hover, etc.) with the button.

Methods #

static load(resource) #

Load the given audio resource. Audio clips must be loaded before playing. Returns a handle, which uniquely identifies this sound source.

static play(handle) #

Play the audio resource indicated by the handle. Resource must have previously been loaded.

static stop(handle) #

Stop playing the audio resource indicated by the handle.

static unload(handle) #

Dispose of any resources associated with this handle.

static setOnReady(handle, callback) #

Assign a callback envoked when the audio is ready to play. The handle is passed as an argument to the callback.

static setOnEnded(handle, callback) #

Assign a callback envoked when the audio is done playing. The handle is passed as an argument to the callback.
